# SpringFramework_JAVA_Configuration :

1) Create The Java Project
2) Configure the Required Jar File(Download From Maven Repositery)
3) Create The Packages
   - for main class
   - for Bean/entity
   - for Java file(repository to set value to bean)
# Work Flow :
- Applicattion Context read Java repository file.
- In Java repository file , value are set to the bean class.
- Using application context bean are get and disply.
